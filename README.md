# ☢️ Bunker Game | Web App Edition

Простое и быстрое веб-приложение для игры в настолку **«Бункер»**. Создано для комфортных партий с друзьями без лишних бумажек и сложных регистраций.

[Посмотреть Live Demo](https://твой-логин.github.io/bunker-game/?job=Разработчик&health=Идеальное&item=Ноутбук+на+Ubuntu)

---

## 🚀 Как это работает?
Приложение работает по принципу **Stateless (без БД)**. Все характеристики персонажа передаются напрямую через URL-параметры. Это позволяет:
* Хостить проект бесплатно на **GitHub Pages**.
* Моментально открывать карточки персонажей из Telegram-бота.
* Не переживать за сохранность данных — ссылка и есть твой персонаж.

## 🛠 Технологический стек
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla).
* **Интеграция:** Telegram Bot API (для генерации ролей).
* **Хостинг:** GitHub Pages.
* **ОС разработки:** Ubuntu 🐧.

## 📦 Структура проекта
* `index.html` — Основной интерфейс карточки игрока.
* `data.json` — (В разработке) База профессий, катастроф и предметов.
* `/assets` — Иконки и стили для атмосферы постапокалипсиса.

## 🎮 Как запустить свою версию
1. Склонируйте репозиторий.
2. Настройте GitHub Pages в настройках репозитория.
3. Сгенерируйте ссылку с параметрами:
   `?job=Инженер&health=Здоров&item=Фонарик`
